Documentation for Cytostream
============================

Welcome to the documentation for Cytostream.  The primary purpose of
this standalone software is to put into the hands of biologists
complex methods for the analysis of flow cytometry data.

Specifically, the methods available are focused on *objectivity* and
*reproducibility*.  The underlying algorithms combined with features
for report production enable users to perform a variety complete analysis
pipelines.

Currently, Cytostream is in an alpha state.  The beta release is
expected within the 2012 year and at that point binaries for all major
operating systems will be provided.

Learning by example
___________________

The documentation for Cytostream focuses on using working examples to
teach the user how to use the various functions available.  Users may
access these function through a programmatic interface (`API
<http://en.wikipedia.org/wiki/Application_programming_interface>`_) or
via the graphical software (`GUI
<http://en.wikipedia.org/wiki/Graphical_user_interface>`_).  It is
also possible to run large projects using the API then visualize the
results in the GUI.
